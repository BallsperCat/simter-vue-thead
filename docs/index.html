Simter thead vue component.

Fork me on Github

The target is to config the table thead by a array config. Use together with simter-vue-colgroup component.

// html
<table style="border-collapse:collapse">
  <colgroup is="st-colgroup" :columns="columns"></colgroup>
  <thead is="st-thead" :columns="columns"
         head-class="head"             -- Optional
         row-class="row"               -- Optional
         cell-class="cell"             -- Optional
         text-overflow="ellipsis">     -- Optional
  </thead>
</table>

Example 1: Simple Columns

// config
columns = [
  { width: "100px", label: "X1",headCellClass: "test" },
  { width: "100px", label: "X2 long text" },
  { width: "100px", label: "X3" }
]

// Result
| X1 | X2 | X3 |

Attention: 'text-overflow' value can be 'ellipsis', 'hidden' or 'word-wrap'.

Example 2: Complex Columns

// config
columns = [
  { width: "100px", label: "X1" },
  {
    width: "100px", label: "X2",
    children: [
      { width: "100px", label: "X21" },
      { width: "100px", label: "X22" }
    ]
  },
  { width: "100px", label: "X3" },
  {
    width: "100px", label: "X4",
    children: [
      {
        width: "100px", label: "X41",
        children: [
          { width: "100px", label: "X411" },
          { width: "100px", label: "X412" }
        ]
      },
      { width: "100px", label: "X42 long text long text long text long text" }
    ]
  }
]

// Result
| X1 |     X2    | X3 |         X4        |
|    | X21 | X22 |    |     X41     | X42 |
|    |     |     |    | X411 | X412 |     |

Attention: Will auto judge the column group by the nested children config.
